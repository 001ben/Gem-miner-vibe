# Gem Miner Bulldozer
A browser-based gem mining game using Matter.js physics.

## How to Play
- **Controls**:
  - Desktop: Use Arrow keys or WASD to move the bulldozer.
  - Mobile: Use the on-screen D-pad.
- **Goal**: Push gems into the collector (the red pad/hole) to earn money.
- **Upgrades**:
  - **Upgrade Dozer**: Increases the size and pushing power of your bulldozer.
  - **Upgrade Collector**: Increases the size of the collector.
  - **Unlock Area**: Expands the playing field.

## Features
- Physics-based movement and collisions.
- custom SVG sprites for bulldozer, gems, and collector.
- Responsive design with mobile touch controls.
